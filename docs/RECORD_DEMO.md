# Record the DevLog Demo

Use this checklist to replace `devlog/media/devlog-demo.png` with a real `devlog/media/demo.gif`.

1. Install the latest VSIX or launch the extension with F5.
2. Open a small sample TypeScript project.
3. Enable `devlog.demoMode` so no Gemini API call is made.
4. Open the DevLog activity bar view.
5. Start screen recording with a tight crop around the editor.
6. Edit one or two files in quick succession.
7. Wait about 2 seconds for one batched lesson to appear.
8. Stop recording and export as `devlog/media/demo.gif`.
9. Replace the README image reference from `devlog/media/devlog-demo.png` to `devlog/media/demo.gif`.

Keep the demo free of API keys, OAuth tokens, private file names, or customer code.
